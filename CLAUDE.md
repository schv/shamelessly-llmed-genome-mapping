# Genome Mapping Project

## Overview
This repository contains work for the CSAI Fall 2025 Algorithm Engineering course, focusing on genome mapping algorithms.

## Project Structure
- Course exercises from Codeforces group
- Genome mapping project using E. coli reference genome

## Key Tasks

### Part 1: Exercises
- Implement algorithms from lectures
- Solve problems from Codeforces outlined in files:
    * `a_task.txt`
    * `b_task.txt`
    * `c_task.txt`
    * `d_task.txt`
    * `e_task.txt`
- Each problem must be solved in a separate single python file using `pypy 3.10` 

### Part 2: Genome Mapping Project
- Map reads to E. coli reference genome
- Algorithms to consider: Suffix arrays, BWT, FM-index, seed-and-extend approaches

## Technology Stack
Use `uv` and `pypy 3.10`

## Data Sources
- Reference genome: https://ftp.ncbi.nlm.nih.gov/genomes/all/GCF/000/005/845/GCF_000005845.2_ASM584v2/GCF_000005845.2_ASM584v2_genomic.fna.gz
- Reads (FASTQ): ftp://ftp.sra.ebi.ac.uk/vol1/fastq/ERR022/ERR022075/ERR022075_1.fastq.gz
